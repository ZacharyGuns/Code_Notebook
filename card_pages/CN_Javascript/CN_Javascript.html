Conditional

Conditionals are If/Else statements. A standard If/Else only has two possible outcomes. Ex: If true, return A. Else return B. If the conditional is true, then we should see A returned to us. However, if the conditional is not true, else false, it will return B instead. In the following example we are using a time conditional. If the time is less than 12 pm, return Good day!
