Conditionals

Conditionals are If/Else statements. A standard If/Else only has two possible outcomes. Ex: If true, return A. Else return B. If the conditional is true, then we should see A returned to us. However, if the conditional is not true, else false, it will return B instead. In the following example we are using a time conditional. If the time is less than 18:00 hours (12am-6pm), return Good day! Else, return Good night! (6pm-12am).

This is an If, Else if, Else coniditional. Conditionals that contain Else if's can have any number of possible outcomes depending on how many Else if's you include. In this case, rather than having only Good day and Good night as possible outcomes for the time, we instead have Good morning (12am-12pm), Good afternoon (12pm-6pm) and Good evening (6pm-12am).

Switch Statement

Switch Statements are evaluated once. The value of the expression is compared with the values of each case. If there is a match, the associated block of code is executed. In this example, there are 7 cases, one for each day of the week. Once the expression is called, it checks which day of the week it currently is and returns the correct one. At the end of each Switch Statement is the "break" keyword. Once the code runs reaches the "break" keyword, it breaks out of the switch block and stops the Switch Statement.

The "default" keyword specifies the code to run if there is no case match. In the following example, the expression is only given two specific cases to search through, Saturday and Sunday. If the current day of the week is neither Saturday nor Sunday, it instead runs the default for all week days Monday through Friday. The statement will return "Today is Saturday" or "Today is Sunday" for each appropriate weekend day. For Monday through Friday, it will return "Looking forward to the weekend."

The following example uses common code blocks. the keys for Saturday and Sunday, Monday-Wednesday and Thursday and Friday all share the same values. If the current day of the week lands on either Saturday or Sunday, it will read and return the same case value, displaying that "It is the weekend." If it is Monday, Tuesday or Wednesday, it will read the same case value and display "Looking forward to the weekend." If it is Thursday or Friday, it will read the same case value and display "Soon it is the weekend." By grouping keys with same values, it helps create "dry" code.
